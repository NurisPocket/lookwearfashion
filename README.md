# 📝 LookWearFashion

소비자의 성향에 따라 계절별로 코디를 자동으로 추천해주는 서비스입니다.

<br/>

## 🧑🏻‍💻 Feature

- 성별, 선호 색상, 옷의 종류를 입력받아 코디 자동 추천
- 추천받은 옷을 바로 구입할 수 있는 사이트로 이동할 수 있는 사이트 제공
- 별점과 후기를 작성 가능

<br/>

## :pencil2: What i take on

- 버튼에 ID값을 부여하여 파일 속에 있는 랜덤 이미지 파일 출력 구현
- 알고리즘을 통한 코디 자동 추천 서비스 구축
